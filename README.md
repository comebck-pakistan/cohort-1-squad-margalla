## 💡 Our Idea

Our squad picked **[Idea 8 — The Social-Commerce Support Agent](https://github.com/Comebck-Pakistan/cohort-1-product-challenge#idea-8--the-social-commerce-support-agent)** from the [Cohort 1 Product Challenge](https://github.com/Comebck-Pakistan/cohort-1-product-challenge). Below are our ideation-stage validation answers, reproduced exactly as we submitted them.

### One specific user

A Pakistani clothing brand owner with 1k to 150k Instagram followers who takes orders manually through WhatsApp and Instagram DMs, has no support staff, and answers every customer message themselves until midnight.

### The painful problem

They receive 80 to 150 DMs daily asking the same things: size availability, COD options, and delivery timelines. They either send a generic auto-reply that answers nothing or manually type each response hours later, by which point the customer has moved on. On top of lost sales, 30 to 35 percent of COD orders return undelivered because customers were never confirmed before dispatch, costing upto PKR 500 per failed return. No existing tool understands Urdu or handles Pakistan's COD workflow.

### Our 8-week wedge

An AI agent that plugs into a clothing brand's Instagram and WhatsApp DMs, reads customer questions in Urdu or Roman Urdu, checks their product catalog, and instantly replies with accurate answers on size, delivery, and COD confirmation.

### The one number that proves people want it

5 Pakistani fashion brand owners using it on real orders, with at least 3 reporting a measurable drop in unanswered messages and COD RTO falling from 30 to 35 percent down to below 20 percent within 30 days.

### Why we picked this idea

We messaged 16 Pakistani clothing brands as customers in one hour and tracked responses live. Only 1 out of 16 replied instantly and completely. The pain is measurable in rupees, the user is one Instagram search away, and no local Urdu-first tool exists. For user research, squad members spoke to people within their circles running online businesses and officially reached out to brands via Instagram and WhatsApp. We have 2 to 3 calls scheduled for next week and are actively reaching more users.

## 🚀 Getting Started — Clone & Contribute

Follow these steps to get this repo on your machine and start contributing code.

### 1. Prerequisites
- Install [Git](https://git-scm.com/downloads)
- A [GitHub account](https://github.com/join) added to the **Comebck-Pakistan** organization
- _(Recommended)_ [VS Code](https://code.visualstudio.com/) and/or the
  [GitHub CLI](https://cli.github.com/)

### 2. Clone the repository
```bash
git clone https://github.com/Comebck-Pakistan/cohort-1-squad-margalla.git
cd cohort-1-squad-margalla
```

### 3. Create a branch for your work
Never commit straight to `main`. Create a branch for each feature or fix:
```bash
git checkout -b your-name/short-description
```

### 4. Make your changes, then stage and commit them
```bash
git add .
git commit -m "Briefly describe what you changed"
```

### 5. Push your branch to GitHub
```bash
git push -u origin your-name/short-description
```

### 6. Open a Pull Request
- Open the repo on GitHub and click **"Compare & pull request"**.
- Describe your change, request a review from a squadmate, and merge once approved.

### 🔄 Keep your local copy up to date
Before starting new work, pull the latest `main`:
```bash
git checkout main
git pull origin main
```
